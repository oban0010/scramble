Welcome to Scramble

The game where you unscramble letter to make words

Instructions

once you start the game, you will be given a scrabled word.
if you correctly guess the word, you will reccive a point. if you guess incorrectly you will reccive a strike. you can also pass a word

Commands

To start new game start()

To make guess use guess()

To skip word use pass pass()

To show the instruction use help()
